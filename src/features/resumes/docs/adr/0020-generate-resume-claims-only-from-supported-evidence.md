# Generate resume claims only from supported evidence

Job-Specific Resume generation may use only Master Resume evidence for resume claims. Fit Requirements with direct or adjacent evidence can drive job-specific emphasis, summary language, skills ordering, and rewritten bullets. Weak or unsupported Fit Requirements can appear in Resume Fit Analysis as cautions or gaps, but they must not be turned into new job-specific claims because doing so would overstate the user's experience.

The complete application resume can still include work-experience bullets that are truthful Master Resume evidence but do not map to a direct or adjacent Fit Requirement. Those bullets preserve the user's work history; they do not claim support for unsupported Job Posting requirements.
